# 部署

这里提供一个 github 静态页的部署方法，先创建仓库 xxx.github.io，xxx 为你 Github 注册的名字，以后 xxx.github.io 将成为你静态页的根域名，配置 Action 自动部署后，写好的代码直接 push 到主分支

# Github Actions

在仓库 Actions 中配置 Workflow，可帮助自动打包程序，git push 后即可触发，它会将打包好的 Dist 文件放入 gh-pages 分支

# Workflow 示例

name: deploy
on:
  push:
    branches: [ main ]
jobs:
  build:
    # 环境
    runs-on: ubuntu-latest
    # 步骤
    steps:
      # 拉代码
      - uses: actions/checkout@v3

      # 初始化 环境
      - name: 初始化 Node.js 环境
        uses: actions/setup-node@v3.1.1
        with:
          node-version: "14.X"

      # 安装依赖
      - name: 安装依赖
        run: npm install

      # 构建 App
      - name: 构建 App
        run: npm run build

      # 部署到 Github Page
      - name: 部署到 Github Page
        uses: peaceiris/actions-gh-pages@v3
        with:
          personal_token: ${{ secrets.PERSONAL_TOKEN }}
          publish_dir: docs/.vuepress/dist
          emptyCommits: false
